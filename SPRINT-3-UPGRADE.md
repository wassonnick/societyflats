# SocietyFlats Sprint 3 Premium Real-Estate Identity Upgrade

This package upgrades the Sprint 2 prototype with:

- Image-led hero section with real estate/lifestyle visual
- Stronger headline and sharper copy
- Larger search experience
- Trust metric cards below hero
- Featured society-first homepage section
- Lifestyle discovery cards
- Rental intelligence mini-chart cards
- Society page storytelling and trend block
- Expanded SEO footer
- Improved card hierarchy and hover states

## Deploy on Vercel

1. Upload this folder/repository to Vercel.
2. Framework preset: Vite.
3. Build command: `npm run build`
4. Output directory: `dist`

## Local Run

```bash
npm install
npm run dev
```
