# Blueprint Lock

This prototype intentionally ignores all previous design experiments.

It follows only:

1. Premium society-first rental positioning
2. Gurgaon + Bangalore only
3. Society pages as moat
4. Search-first homepage
5. Verified inventory trust layer
6. Future-ready tenant, owner, broker, admin architecture
7. Clean design system using navy, warm white, gold, restrained typography
8. Component-first React frontend
